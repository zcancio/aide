# AIde CLI

Terminal REPL for talking to AIde.

## Installation

```bash
pip install aide-cli
```

## Usage

```bash
# Authenticate
aide login

# Start REPL
aide

# REPL commands
/list       - Show all aides
/switch <n> - Switch to aide
/new        - Start new aide
/page       - Open published page
/info       - Show current aide details
/history    - Show message history
/quit       - Exit
```

## Configuration

Config stored at `~/.aide/config.json`
